# Moving Object Detection and Trajectory Tracking

Phase 1: Moving Object Detection using MOG2

## Installation

```bash
pip install -r requirements.txt
```

## Running

1. Place your video in `dataset/VIRAT/` (e.g. `video1.mp4`).
2. Run the main script from the root project directory:
```bash
python src/main.py
```
```bash
python src/main.py --video dataset/VIRAT/your_video.mp4
```
